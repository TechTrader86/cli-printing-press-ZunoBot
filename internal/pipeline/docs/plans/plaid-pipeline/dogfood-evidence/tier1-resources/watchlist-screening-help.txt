Manage watchlist screening

Usage:
  plaid-cli watchlist-screening [command]

Available Commands:
  entity-create            Create a watchlist screening for an entity
  entity-get               Get an entity screening
  entity-history-list      List history for entity watchlist screenings
  entity-hit-list          List hits for entity watchlist screenings
  entity-list              List entity watchlist screenings
  entity-program-get       Get entity watchlist screening program
  entity-program-list      List entity watchlist screening programs
  entity-review-create     Create a review for an entity watchlist screening
  entity-review-list       List reviews for entity watchlist screenings
  entity-update            Update an entity screening
  individual-create        Create a watchlist screening for a person
  individual-get           Retrieve an individual watchlist screening
  individual-history-list  List history for individual watchlist screenings
  individual-hit-list      List hits for individual watchlist screening
  individual-list          List Individual Watchlist Screenings
  individual-program-get   Get individual watchlist screening program
  individual-program-list  List individual watchlist screening programs
  individual-review-create Create a review for an individual watchlist screening
  individual-review-list   List reviews for individual watchlist screenings
  individual-update        Update individual watchlist screening

Flags:
  -h, --help   help for watchlist-screening

      --timeout duration   Request timeout (default 30s)
